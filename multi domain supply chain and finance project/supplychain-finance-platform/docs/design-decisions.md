# ADR: Architecture Decision Records

## ADR-001: Microservices Architecture
- **Status**: Accepted
- **Context**: Need for scalable, maintainable system architecture
- **Decision**: Adopt microservices architecture
- **Consequences**: Better scalability but increased complexity

## ADR-002: Tech Stack Selection
- **Status**: Accepted
- **Context**: Need to choose primary technologies
- **Decision**: 
  - Backend: Node.js/Python microservices
  - Frontend: React
  - Database: PostgreSQL + MongoDB
  - Blockchain: Hyperledger Fabric
  - AI/ML: TensorFlow/PyTorch
- **Consequences**: Modern stack with good community support
